# Sistema de Registro de Participantes

## Descripción
Aplicación Java con interfaz gráfica para capturar y presentar datos de participantes.

## Archivos principales
- `ProcesarDatos.java` - Clase que maneja la lógica de datos
- `InterfazParticipante.java` - Interfaz gráfica del usuario

## Cómo ejecutar
1. Abrir el proyecto en NetBeans
2. Ejecutar `InterfazParticipante.java`
